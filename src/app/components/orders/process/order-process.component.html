1.

 

The Order Processing Procedures can be found here for your reference. Please make sure you have reviewed them before you continue with processing of the order, as is is important that every order is processed accuractely and in a similar fashion for a good customer experience.

2.

 

The Order Invoice is the blue-print for the order. It contains not only all of the items and quantities that are on the order, but also the mailing address needed to generate the shipping label. Make sure you have a copy of it printed out so that you can use it to process the order accurately.
